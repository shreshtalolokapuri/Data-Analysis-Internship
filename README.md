# Data-Analysis-Internship
Data Analytics Internship Project | Data Cleaning, SQL Analysis, Interactive Dashboards &amp; Business Insights
  
  
**Overview**  

This repository showcases my Data Analytics Internship project, where I performed end-to-end data analysis to transform raw data into actionable business insights. The project involved data cleaning, SQL-based analysis, dashboard development, and business reporting.  
  
**Objectives**  
  
- Clean and organize raw data for analysis.  
- Analyze business data using SQL queries.  
- Create interactive dashboards to visualize key metrics.  
- Generate actionable insights to support business decision-making.  
  
**Tools & Technologies**  
  
- Microsoft Excel – Data cleaning, preprocessing, pivot tables, and charts  
- SQLite (Online) – SQL querying and data analysis  
- Power BI – Interactive dashboard creation  
- Microsoft Word – Business insights and recommendations report  
  
**Project Workflow**  
  
1. Cleaned and organized the dataset using Excel.  
2. Performed business analysis using SQL queries.  
3. Built an interactive Power BI dashboard to visualize trends and patterns.  
4. Documented key findings and business recommendations in a report.
  
**Repository Structure**  
  
- Dataset – Source dataset  
- Excel – Cleaned dataset, pivot tables, and charts  
- SQL – SQL queries used for analysis  
- PowerBI – Power BI dashboard  
- Report – Business insights and recommendations  
- Images – Dashboard screenshots  
  
**Key Skills Demonstrated**  
  
- Data Cleaning & Preparation  
- SQL Querying  
- Data Analysis  
- Dashboard Development  
- Business Insight Generation  
- Data Visualization  
- Analytical Thinking
